# 语音对话应用 - 开发文档

## 1. 项目概述

**项目名称**：VoiceChat

**核心功能**：手机按压录音 → 在线 STT 转文字 → 发送到 OpenClaw 服务器 → 接收文字回复 → 显示

**目标用户**：银政

**技术选型**：
- 手机端：Android 原生（Java 17，API 26+）
- 服务器端：Node.js + Express
- 通信：HTTP REST API

---

### Android 端

|------|------|--------|------|

## 3. 技术方案

### Android 端

**开发环境**：Android Studio Iguana+ / Java 17 / Gradle 8.7 / minSdkVersion: 26，targetSdkVersion: 34

**服务器地址**：
- 公网：`http://123.58.210.65:18881`
- 内网：`http://10.7.7.180:18881`

**STT 方案**：
- 主用：SiliconFlow FunAudioLLM/SenseVoiceSmall（在线识别，中文支持好）
- 已移除本地 Vosk 离线识别

**依赖**：
- androidx.appcompat:appcompat:1.6.1
- com.google.android.material:material:1.11.0
- androidx.constraintlayout:constraintlayout:2.1.4
- com.squareup.okhttp3:okhttp:4.12.0
- com.google.code.gson:gson:2.10.1

### 服务器端

**环境**：Node.js v22 + Express / 端口：18881

**接口**：
```
POST /api/chat
Request: { "text": "用户说的话", "sessionId": "可选" }
Response: { "code": 0, "reply": "助手回复", "sessionId": "会话ID" }

POST /api/audio
Request: binary audio (ogg/wav/m4a)
Response: { "code": 0, "text": "识别结果" }

POST /api/upload
Request: multipart/form-data { "file": <binary>, "type": "image|audio|file", "sessionId": "可选" }
Response: { "code": 0, "url": "/api/download/xxx", "fileId": "xxx" }

GET /api/download/<fileId>
Response: binary file

GET /api/audio/<fileId>
Response: audio/mpeg binary
```

**AI 模型**：minimax/MiniMax-M2.7
